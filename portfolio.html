OpenWeather Project

Aplicación que permite visualizar datos meteorológicos utilizando la API de OpenWeatherMap. Incluye pruebas automatizadas, arquitectura modular y componentes de Android modernos.

  • ⚙️ Integración con Retrofit para llamadas a API.
  • 💾 Base de datos local con Room para persistencia offline.
  • 🧪 Pruebas unitarias con Mockito, MockWebServer y Room Testing.
  • 🎨 UI interactiva diseñada con Material Design.
